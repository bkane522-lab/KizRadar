KIZRADAR — Reprise de contrôle (retour à notre version)
=========================================================

Contexte : une session ChatGPT a modifié le projet en parallèle
(nouveau schéma de données, système de curation automatique).
Ce zip restaure notre version : soumission communautaire simple
+ page d'admin manuelle, que tu maîtrises entièrement.

FICHIERS DE CE ZIP (tout remplace l'existant) :
- index.html       -> racine
- sw.js            -> racine
- manifest.json    -> racine
- api/events.js    -> dossier api/ (remplace le fichier de ChatGPT)
- admin.html       -> racine, NOUVEAU fichier (page de modération)

MARCHE À SUIVRE
1) Colle chacun de ces 5 fichiers dans GitHub (contenu complet,
   pas de fusion avec l'ancien).
2) Vercel > kiz-radar > Deployments > Redeploy.
3) Vérifie que https://kiz-radar.vercel.app/api/events renvoie
   {"events":[]} (base vide, prête pour tes vraies soirées).
4) Va sur https://kiz-radar.vercel.app/admin.html
   Clé : kizradar-clean-2026
   -> tu dois voir la liste (vide pour l'instant) + bouton
      Supprimer sur chaque future soirée.
5) Ajoute tes soirées via le bouton "+" de l'app, ou signale-moi
   quand c'est en ligne, je t'aide à ajouter les vraies soirées
   parisiennes (+ les bonnes trouvailles de ChatGPT si tu veux
   les garder, ajoutées manuellement dans notre système).

RAPPEL SÉCURITÉ
La clé admin "kizradar-clean-2026" ne doit être connue que de toi.
Elle protège la suppression d'événements sur /api/events et sur
la page /admin.html.
